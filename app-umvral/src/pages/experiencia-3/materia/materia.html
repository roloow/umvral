Descripción de Temperatura y dilatación

Se considera como dilatación térmica el fenómeno que ocurre cuando las dimensiones de un cuerpo cambian según la temperatura a la cual está expuesto, es decir, un cuerpo puede aumentar o disminuir su Largo, Área o Volumen según la variación de temperatura que exista.

Dependiendo de las dimensiones a estudiar, el fenómeno anteriormente explicado se llama dilatación lineal, superficial o volumétrica. Si sólo se estudia una dimensión, por ejemplo el eje x, fenómeno se llama dilatación lineal, puesto que la variación será sólo estudiada en el largo del objeto. Si son dos variables las de estudio, por ejemplo, x e y, se llama dilatación superficial, puesto que ahora se estudia la variación del área o superficie en respecto al tiempo. Finalmente, si son consideradas las 3 dimensiones, llamese z, x e y, se conoce como dilatación volumétrica, puesto que la variación que se estudia es la que presenta el volumen del objeto.

Identificando como L el largo de un objeto, A el área y V el volumen, además asignando los coeficientes de dilatación como λ para la dilatación lineal, σ para la dilatación superficial y γ para la dilatación volumétrica, en conjunto con Δ para hablar de variación, se tienen las siguientes ecuaciones:

γ = 3λ, σ = 2λ
ΔT = TFinal-TInicial
ΔL = L0 (1 + λ ΔT)
ΔA = A0 (1 + σ ΔT)
ΔV = V0 (1 + γ ΔT)

De las ecuaciones anteriores, se ve que si el coeficiente de dilatación es mayor a 0, cuando la temperatura aumenta, el objeto de estudio crece, mientras que si disminuye, el cuerpo disminuye, ocurriria lo contrario si el coeficiente fuese menor a cero.
